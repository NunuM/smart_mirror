Smart Mirror Project

## QT - UI 
The GUI was made with QT framework. The comunication is done via DBUS. Several interfaces are exposed via DBUS.

#### Widgets implemented  

* Applications: list of widgets that are installed.
* Media: list of movies.
* Weather
* Traffic
* Notes
* Reddit
